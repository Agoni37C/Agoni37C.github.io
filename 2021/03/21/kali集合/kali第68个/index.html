❄Kali linux 学习笔记（六十八）隧道——ptunnel 2020.4.11

flechazo · 2021-3-21 · 次阅读








ptunnel


  • ping tunnel icmp 隧道工具
  • 通过ICMP的echo (ping requests)和reply(ping reply) 实现隧道
  • 适用于防火墙只允许ping出站流量的环境
  • 支持多并发连接、性能优
  • 支持身份验证
  • 需要root权限
  • 似乎存在漏洞
  • 支持抓包：需要winpcap或libpcap
  • 工作过程如图所示


用户和代理用ICMP建立隧道，然后通过代理用TCP访问目标


路由器192.168.1.1


防火墙m0n0wall
桥接：192.168.1.10
hostonly：1.1.1.1
设置如图所示


服务器kali
桥接：192.168.1.14


ptunnel -x pass123 #指定连接密码

    客户端 xubuntu
    hostonly：1.1.1.12


    ptunnel -p 192.168.1.14 -lp 8000 -da 192.168.1.1 -dp 80 -x pass123
    # -p：服务端 IP，代理
    # -lp：启动本地监听端口
    # -da：目标 IP
    # -dp：目标 Port
    # -x：连接密码
    #以上命令指示打开监听端口，当有流量访问时，才会建立隧道
    #嵌套 SSH 隧道
    ptunnel -p 192.168.1.14 -lp 2222 -da 192.168.1.14 -dp 22 -x pass123
    ssh -CNfg -D 7000 root@127.0.0.1 -p 2222
