❄Kali linux 学习笔记（一）安装与基础配置 2020.2.11

flechazo · 2020-2-11 · 次阅读








前言


被疫情封闭在乡下，出不了门，出于兴趣开始学习Kali Linux
是安全牛的教程，总计177课，每课时1-2h，预计三个月内学完
完全从零开始，迫不及待想获取知识
以上


先插播一个笔者觉得很有意思的问题
kali命名的由来
一个有意思的解释如下：（笑）


一、安装


由于网上安装教程很多且很详细，csdn里随意一搜就有，故不做展开，仅简单讲述过程：


  • 登陆Kali Linux 官网下载映像文件（64位or32位根据需要）笔者是2020版64位
  • 下载VMware虚拟机，笔者是vm15
  • 在虚拟机上安装完成kali linux
  • 为了能与物理机复制黏贴文件，虚拟机要安装下VMware tools
    在2020.2.13发现一篇详细安装教程，在此补上：
    https://blog.csdn.net/weixin_41247970/article/details/104118700

值得注意的是，kali linux可能出现中文乱码的情况
笔者就遇到了
网上众说纷纭
笔者最终解决方案是
执行sudo apt-get install ttf-wqy-zenhei后重启即可


二、配置环境


1、更新源


第一件事是换更新源
文件是/etc/apt/sources.list
更改方式


vi /etc/apt/sources.list

    按i进入插入模式
    从网上找合适的源，如阿里云的源
    复制过来


    :wq

      保存退出


      然后执行


      apt-get update && apt-get upgrade

        意思是依次从源更新软件的索引文件和软件本身
        这一步通常每次打开kali linux都要做，因为它更新很快


        2、中文输入法


        中文输入法很重要嘛对不对
        网上有些安装搜狗或谷歌的输入法，我试了下，可能因为版本问题，不太靠谱
        还是老牌的ibus靠谱
        执行


        apt-get install ibus ibus-pinyin              #经典的ibus

          就安装好了
          然后从菜单找到ibus点开进行相关配置就好
          重启就可用


          三、一些linux指令


          放些linux指令在这里


          # 查看文件
          ls #显示可见文件
          ls -a #隐藏文件
          ls -l #详细信息
          ls -lh #详细信息友好版

          #进入目录
          cd + （文件夹路径）
          cd. #当前
          cd.. #上级目录
          pwd #查看当前目录

          #文件内容
          cat + （文件路径）
          more + （文件路径）#区别是显示一页停住
          tail -（数字：想看最后几行） + （文件路径）#看最后几行
          grep +（特定字符串）+（文件路径）#看特定的信息
          退出是q
          #循环往复 watch -n （时间）

          #文件操作
          rm + （文件名）#删除文件
          rm + （文件夹名）#删除文件夹
          cp + （文件名）+（想得到的文件名） #复制文件
          cp -r + （文件夹名）+（想得到的文件夹名） #复制文件夹

          #监视系统
          top
          kill + （pid）#删除进程
          q #退出

          #查看进程
          ps
          ps -ef #详细版
          ps aux #另一种详细版


          #显示信息
          echo “（内容）”#在当前终端显示内容
          echo -n “（内容）”#光标不换行

          #编辑文件
          #第一种
          vi （文件名）#编辑文件
          :set nu #显示行号
          :wq #保存退出
          i #插入模式
          dd #删除当前行
          o #插入一行
          y #复制当前行
          p #黏贴
          #第二种
          mousepad （文件名） #之前的linux版本是leafpad


          #命令之间
          & #依次执行两个命令
          && #前一个命令成功后执行后一个
          | #前一个命令的输出作为后一个命令的输入
          || #前一个命令不成功就执行后一个

          # 执行文件
          ./（文件）

          安装和基础配置就这样，下一章进行环境优化
